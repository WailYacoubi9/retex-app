OPÉRATEURS : "=", "!=", "contient", ">=", "<=", "annee" (champ date + valeur AAAA),
"mois" (champ date + valeur AAAA-MM), "est_rempli", "est_vide".

RÈGLES :
- intent : "count" si la question demande un nombre ("combien"), "repartition" si
  elle demande une répartition/ventilation/distribution (alors group_by = le champ),
  "non_supporte" si la question demande un calcul indisponible (somme, moyenne,
  coût, durée) ou porte sur une donnée absente des champs listés,
  "liste" sinon.
- booléens : op "=" et valeur "true" ou "false".
- présence d'une info : op "est_rempli". Absence : "est_vide".
- dates : "annee" avec valeur "2025", "mois" avec "2025-03", ou ">="/"<=" avec "AAAA-MM-JJ".
  Dates RELATIVES : calcule à partir de DATE DU JOUR ("hier" → op "=" et la date
  d'hier ; "cette année" → op "annee" et l'année courante).
  DATE DU JOUR sert UNIQUEMENT aux dates relatives : n'ajoute JAMAIS de filtre
  sur l'année courante si la question ne la mentionne pas.
  PLAGE ("entre 2023 et 2025", "de X à Y", "évolution entre...") → DEUX filtres :
  op ">=" valeur "2023-01-01" ET op "<=" valeur "2025-12-31".
- champs à choix (échelles) : choisis la valeur dont le libellé ou le SENS
  correspond aux mots de la question. Ne saute JAMAIS à un autre niveau d'échelle.
- entités liées : op "contient" avec le nom.
- N'ajoute JAMAIS un filtre non demandé dans la question. Aucun filtre implicite.
- group_by : le NOM EXACT d'un champ (jamais de parenthèses ni d'expression),
  ou "annee" / "mois" pour une répartition temporelle. Null hors répartition.
- "moyenne", "somme", "coût", "durée moyenne" → intent non_supporte.

EXEMPLES GÉNÉRIQUES (question → spec) :
- "Combien de fiches en 2024 ?" → intent count, filtres [champ date_evenement, op annee, valeur 2024]
- "Les fiches avec une action à chaud" → intent liste, filtres [champ action_corrective, op est_rempli]
- "Répartition par condition lumineuse" → intent repartition, group_by condition_lumineuse, filtres []
- "Combien de fiches hier ?" → intent count, filtres [champ date_evenement, op =, valeur {hier}]
- "Quelle est la moyenne des coûts ?" → intent non_supporte
