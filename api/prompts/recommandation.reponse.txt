« {description} »

{contexte}

- Commence par une phrase indiquant que des incidents similaires existent (cite 2-3 FE).
- Puis présente les actions par catégorie, dans cet ordre : actions PRÉVENTIVES
  (les plus utiles pour éviter la récidive), actions CORRECTIVES, actions À CHAUD
  (réactions immédiates du terrain).

- Si une action revient sur plusieurs incidents, souligne-le : c'est un signe de pratique établie.
- Termine par une phrase de synthèse sur la ou les actions les plus pertinentes.
- N'invente RIEN : uniquement les actions listées ci-dessus.
